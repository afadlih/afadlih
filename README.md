## About

I am a D4 Informatics Engineering student focused on building AI-powered web applications with a product-oriented approach. My work goes beyond generating AI output — I design systems that can evaluate options, compare strategies, and support structured decision-making.

**Core areas:**
- Fullstack Development — Next.js, TypeScript, Tailwind CSS
- AI Integration — Gemini API, structured prompt engineering
- AI Decision Systems — generate → evaluate → compare → decide → execute

---

## What I Build

```
Input
  └── AI Generation      multiple strategies or outputs
        └── Evaluation   scoring per dimension (engagement, hook, CTA)
              └── Comparison   decision matrix across options
                    └── Decision     select best with reasoning
                          └── Optimization   refine for execution
                                └── Output   structured, reliable, actionable
```

Every system I ship includes response normalization, fallback handling, and error recovery — not just the happy path.

## Projects

### AI Content Strategy & Decision System

An AI pipeline that generates multiple content strategies, scores each one, and selects the best option — automatically.

```
Strategies   emotional / educational / viral
Evaluation   engagement score · hook quality · CTA strength
Decision     weighted matrix → best strategy with reasoning
Reliability  response normalization + fallback handling (Gemini API)
```

Stack: `Next.js` `TypeScript` `Tailwind CSS` `Gemini API`

---

### Smart Clothesline IoT System

An IoT automation system that controls a clothesline based on real-time environmental conditions.

```
Monitoring   real-time sensor dashboard (Next.js)
Integration  sensor data → web system
Automation   environment-condition-based logic
```

Stack: `Next.js` `TypeScript` `IoT Sensors`
